## Sobre o projeto
Este projeto é uma SPA desenvolvida em Laravel 8 e Vue.js 2.

## Dependências
- Vuex
- Vue-router
- Vuetify
- Vue-snotify
- Telescope

## Executando o projeto

- npm install
- npm run dev

Em outro terminal, executar os comandos abaixo:
- docker-compose up -d
- docker-compose exec -it my_app bash
- cp .env.example .env
- composer install
- php artisan key:generate
- php artisan db:seed
